Hash table은 키를 값에 매핑할 수 있는 구조인,
연관 배열 추가에 사용되는 자료 구조이다.
Hash table은 Hash 함수를 사용하여 색인(index, Key)을
버킷(bucket)이나 슬롯(slot)의 배열로 계산한다.
주어진 N개의 key, data쌍을 Hash table에 입력한 후,
Q개의 key를 입력 받아 key에 해당하는 data를 각 줄에 출력하시오.
(1<=N, Q<=4096)
